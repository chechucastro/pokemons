# Pokemon

A Vite frontend project to fetch and display Pokemon information.

## Production URL

[https://pokemons-6su.pages.dev/](https://pokemons-6su.pages.dev/)

## Requirements

- Node.js 18+ (recommended)
- npm

## Installation

```bash
npm install
```

## Local development

```bash
npm run dev
```

## Production build

```bash
npm run build
```

## Local preview of the build

```bash
npm run preview
```
