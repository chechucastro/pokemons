# Pokemon

Proyecto frontend con Vite para consultar y visualizar informacion de Pokemon.

## URL de produccion

[https://pokemons-6su.pages.dev/](https://pokemons-6su.pages.dev/)

## Requisitos

- Node.js 18+ (recomendado)
- npm

## Instalacion

```bash
npm install
```

## Desarrollo local

```bash
npm run dev
```

## Build de produccion

```bash
npm run build
```

## Preview local del build

```bash
npm run preview
```
